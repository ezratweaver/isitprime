# Is It Prime?

A simple but effective program to check if a number is prime, using the tkinter GUI library.


![checking_numbers](https://user-images.githubusercontent.com/101545981/233709144-efff4382-e10f-49e6-ab5a-4f2c72ecb71e.gif)



### Now with themes!

Now choose between light or dark mode! It will also detect the theme you are using on windows and adjust accordingly! Checking for prime numbers has never looked so good before.
